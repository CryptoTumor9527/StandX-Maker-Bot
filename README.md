## 简介

StandX 做市机器人是一款运行在本地终端的高频自动交易程序。它通过直连 StandX Perps API，实现毫秒级的挂单做市。

StandX 更新了API，Token和私钥可以直接生成，不需要在浏览器调用，所以浏览器版本失效了。

下载本项目下的 **bot.js** 和 **package.json** 文件即可运行。



## 安装说明

1. **环境要求**

   需要安装
   - Node.js (建议 v16 或更高)
   - npm (Node 包管理器)

3. **安装依赖**
   在项目目录下终端执行：
   ```
   npm install
   ```
   (主要依赖: `@noble/curves`, `uuid`, `bs58`, `node-fetch`)

## 使用指南

1. **启动机器人**
   在项目目录下终端执行：
   ```
   npm start
   ```

3. **配置参数**
   启动后，按照中文提示输入以下参数 (括号内为默认值)：

   - **Token**: 现在可以在交易所直接生成。
   - **私钥**: 现在可以在交易所直接生成。
   - **杠杆倍数 (10)**: 建议 5-20倍，根据风险偏好设置。
   - **单笔价值 (2000)**: 每次挂单的名义价值 (DUSD)。
   - **做单方向 (long)**: `long` (做多), `short` (做空), `both` (双向).
   - **价格偏移 (0.002)**: 重新挂单的阈值 (0.002 = 0.2%).
   - **检查间隔 (500)**: 循环检测的间隔毫秒数，越小反应越快。
